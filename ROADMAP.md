# 🗺️ ROADMAP.md - Kamisama Mod Loader Roadmap

## 🏁 Phase 1: Core Foundation ✅
- [x] Electron + React 19 scaffolding.
- [x] Basic mod listing and extraction logic.
- [x] IPC bridge with strict typing.

## 🚧 Phase 2: User Experience (Current)
- [ ] **Custom Protocols**: Full implementation of `kamisama://` for browser-to-app mod downloads.
- [ ] **Mod Profiles**: Allow users to save and switch between different mod configurations.
- [ ] **Visual Overhaul**: Implement advanced Glassmorphism and micro-interactions.
- [ ] **Auto-Updater**: Secure background updates for the loader.

## 🚀 Phase 3: Advanced Features (Future)
- [ ] **Conflict Detection**: Identify and warn users about mods that overwrite the same files.
- [ ] **Cloud Sync**: Optional cloud backup for mod profiles and settings.
- [ ] **Community Integration**: Direct browsing of popular mod sites within the app.

## 🎯 v1.0 Release
A 100% functional, secure, and beautiful mod management experience for the Dragon Ball community.
